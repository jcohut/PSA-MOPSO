Testing mopsocd(Multi-objective Particle Swarm Optimization with Crowding Distance) and mopsopsa algorithm v.1

Here are results of running mopsocd and mopsopsa_v1 on testing functions.

Load algorithms, test functions and utilities:

  source('../mopsocd.R')
  source('../mopsocds.R')
  source('../moea.test.dtlz4.R')
  source('../mopsopsa_v1.R')

Set number of points to be generated for True Pareto Front:

  pfpoints <- 10000

Visualizations of solutions MOPSOCD vs MOPSOPSA_V1

Color conventions: Black = True Pareto front, Green = Nondominated solutions, Red = All stored solutions
